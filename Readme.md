# Site Pessoal

Site pessoal desenvolvido para apresentar quem eu sou, minhas intenções profissionais e minha visão como desenvolvedor, indo além de um portfólio tradicional.

## Sobre o Projeto

Este projeto foi criado com o objetivo de apresentar não apenas meus trabalhos, mas também minha identidade profissional como programador.

Mais do que listar projetos, este site busca mostrar o desenvolvedor por trás do código: **Pedro**.  
Aqui, apresento minha forma de pensar, meus valores, minha abordagem em desenvolvimento de software e meus objetivos de carreira.

O foco do projeto é criar uma conexão mais clara entre minhas habilidades técnicas e quem eu sou como profissional, oferecendo uma visão mais completa do meu perfil.

## Versão Online

O projeto está disponível online e pode ser acessado diretamente pelo navegador, sem necessidade de instalação local.

## Tecnologias Utilizadas

- HTML
- React
- Tailwind CSS
- Vite
- Vercel (deploy)

## Organização do Projeto

Este site integra um portfólio desenvolvido anteriormente, acessado por meio de links (`href`).  
Essa abordagem permite reaproveitamento de código, manutenção mais simples e uma organização clara do projeto final.

## Status

Projeto em evolução contínua, acompanhando meu desenvolvimento técnico e profissional.
